二叉树和多叉树的遍历

发表于 2017-04-17

简要：实现二叉树的前序、中序、后序遍历，多叉树的深度优先和广度优先遍历

需要具备的基础知识

JavaScript，树的数据结构

说明

在浏览器中，一切元素都是由标签作为根节点发展成的树,对DOM的操作实质上就是对树的操作。

例1.实现一个如图所示的二叉树，以动画形式对其进行前中后序遍历，遍历到的节点颜色改变

分析：

  1. 使用flex布局DOM，展现出页面上的二叉树样式
  2. 给按钮绑定点击事件
  3. 对树进行遍历，将结果按顺序存入数组
  4. 以动画形式渲染数组，逐个改变背景色
